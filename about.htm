About GreenFlix

GreenFlix is a demo movie browser built with Bootstrap and TMDB. Search movies, browse genres, see trailers, and find where to watch — all in a clean, green UI.

  • Responsive Bootstrap layout
  • Genre filters & pagination
  • Trailer modal (YouTube/TMDB)
  • Watch providers (links to legal services)

This product uses the TMDB API but is not endorsed or certified by TMDB.
